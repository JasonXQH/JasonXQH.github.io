жҜҸдёҖдёӘJavaзЁӢеәҸйғҪеҝ…йЎ»жңүдёҖдёӘmainеҮҪж•°пјҢд№ҹе°ұжҳҜдё»еҮҪж•°

JavaдёӯпјҢиҝҷдәӣеҮҪж•°дёҚдјҡеҚ•зӢ¬еӯҳеңЁпјҢд»–д»¬йғҪеңЁзұ»йҮҢйқўгҖӮеҪ“еҮҪж•°ж”ҫеңЁзұ»дёӯж—¶пјҢжҲ‘д»¬з§°е…¶дёәзұ»зҡ„ж–№жі•гҖӮеңЁJavaдёӯпјҢзұ»е’Ңж–№жі•йғҪеә”иҜҘе…·жңүи®ҝй—®дҝ®йҘ°з¬Ұ(access modifier)пјҢи®ҝй—®дҝ®йҘ°з¬ҰжҳҜprivateпјҢpublicд№Ӣзұ»зҡ„.

еңЁJavaдёӯпјҢжҲ‘д»¬з”ЁдёҚеҗҢзҡ„ж–№жі•жқҘе‘ҪеҗҚжҲ‘д»¬зҡ„зұ»е’ҢжҲ‘д»¬зҡ„ж–№жі•гҖӮ

How Java Code Gets Executed

еҪ“жҲ‘д»¬и·‘JavaзЁӢеәҸзҡ„ж—¶еҖҷпјҢжҳҜжҖҺд№ҲиҝҗдҪңзҡ„гҖӮ

еңЁзј–иҜ‘жӯҘйӘӨдёӯпјҢIntellij дҪҝз”ЁJavaзј–иҜ‘еҷЁжҠҠд»Јз Ғзј–иҜ‘жҲҗJava bytecodeзҡ„еҪўејҸгҖӮзј–иҜ‘е®ҢжҲҗеҗҺпјҢJava bytecodeеҸҜд»ҘеңЁWindows пјҢMacпјҢLinuxжҲ–д»»дҪ•жңүJREпјҲJava runtime environmentпјүзҡ„ж“ҚдҪңзі»з»ҹдёҠиҝҗиЎҢгҖӮ JREдёӯеҗ„дёӘж“ҚдҪңзі»з»ҹйғҪжңүдёҖдёӘиҪҜд»¶еҸ«еҒҡJVM(Java Virtual Machine) .JVM дјҡжҠҠJava byte code иҪ¬жҚўжҲҗ WindowsеҸҜд»ҘзҗҶи§Је№¶жһ¶жһ„зҡ„Native CodeгҖӮиҝҷе°ұжҳҜжүҖи°“зҡ„ дёҖж¬Ўзј–иҜ‘пјҢеҲ°еӨ„иҝҗиЎҢ

javaжңү4дёӘзүҲжң¬пјҡJava SEпјҢJava EEпјҢJava ME ,Java Card

Java SE жҳҜJava Standard Editionзҡ„зј©еҶҷпјҢиҝҷжҳҜJavaзҡ„ж ёеҝғе№іеҸ°пјҢеҢ…еҗ«дәҶJavaзҡ„жҜҸдёҖдёӘеә“

Java EE жҳҜJava Enterprise Editionзҡ„зј©еҶҷпјҢз”ЁдәҺжһ„е»әи¶…еӨ§еһӢгҖҒеҲҶеёғејҸзі»з»ҹгҖӮе®ғе»әз«ӢеңЁSEд№ӢдёҠ

Java MEжҳҜJava Micro Editionзҡ„зј©еҶҷпјҢиҝҷжҳҜдё“й—Ёдёәз§»еҠЁи®ҫеӨҮи®ҫи®Ўзҡ„Javaж ҮеҮҶзүҲзҡ„еӯҗйӣҶ

Types

Primitive Types

Primitive TypesжҳҜеӯҳеӮЁsimple valueзҡ„пјҢд№ҹе°ұжҳҜдёӢйқўзҡ„е…«з§Қеҹәжң¬зұ»еһӢ

JavaиҜӯиЁҖжҸҗдҫӣдәҶе…«з§Қеҹәжң¬зұ»еһӢгҖӮе…ӯз§Қж•°еӯ—зұ»еһӢпјҲеӣӣдёӘж•ҙж•°еһӢпјҢдёӨдёӘжө®зӮ№еһӢпјүпјҢдёҖз§Қеӯ—з¬Ұзұ»еһӢпјҢиҝҳжңүдёҖз§Қеёғе°”еһӢгҖӮ

byteпјҡ

  • byteж•°жҚ®зұ»еһӢжҳҜ8дҪҚгҖҒжңүз¬ҰеҸ·зҡ„пјҢд»ҘдәҢиҝӣеҲ¶иЎҘз ҒиЎЁзӨәзҡ„ж•ҙж•°пјӣ
  • жңҖе°ҸеҖјжҳҜ-128пјҲ-2^7пјүпјӣ
  • жңҖеӨ§еҖјжҳҜ127пјҲ2^7-1пјүпјӣ
  • й»ҳи®ӨеҖјжҳҜ0пјӣ
  • byteзұ»еһӢз”ЁеңЁеӨ§еһӢж•°з»„дёӯиҠӮзәҰз©әй—ҙпјҢдё»иҰҒд»Јжӣҝж•ҙж•°пјҢеӣ дёәbyteеҸҳйҮҸеҚ з”Ёзҡ„з©әй—ҙеҸӘжңүintзұ»еһӢзҡ„еӣӣеҲҶд№ӢдёҖпјӣ
  • дҫӢеӯҗпјҡbyte a = 100пјҢbyte b = -50гҖӮ

shortпјҡ

  • shortж•°жҚ®зұ»еһӢжҳҜ16дҪҚгҖҒжңүз¬ҰеҸ·зҡ„д»ҘдәҢиҝӣеҲ¶иЎҘз ҒиЎЁзӨәзҡ„ж•ҙж•°
  • жңҖе°ҸеҖјжҳҜ-32768пјҲ-2^15пјүпјӣ
  • жңҖеӨ§еҖјжҳҜ32767пјҲ2^15 - 1пјүпјӣ
  • Shortж•°жҚ®зұ»еһӢд№ҹеҸҜд»ҘеғҸbyteйӮЈж ·иҠӮзңҒз©әй—ҙгҖӮдёҖдёӘshortеҸҳйҮҸжҳҜintеһӢеҸҳйҮҸжүҖеҚ з©әй—ҙзҡ„дәҢеҲҶд№ӢдёҖпјӣ
  • й»ҳи®ӨеҖјжҳҜ0пјӣ
  • дҫӢеӯҗпјҡshort s = 1000пјҢshort r = -20000гҖӮ

intпјҡ

  • intж•°жҚ®зұ»еһӢжҳҜ32дҪҚгҖҒжңүз¬ҰеҸ·зҡ„д»ҘдәҢиҝӣеҲ¶иЎҘз ҒиЎЁзӨәзҡ„ж•ҙж•°пјӣ
  • жңҖе°ҸеҖјжҳҜ-2,147,483,648пјҲ-2^31пјүпјӣ
  • жңҖеӨ§еҖјжҳҜ2,147,483,647пјҲ2^31 - 1пјүпјӣ
  • дёҖиҲ¬ең°ж•ҙеһӢеҸҳйҮҸй»ҳи®Өдёәintзұ»еһӢпјӣ
  • й»ҳи®ӨеҖјжҳҜ0пјӣ
  • дҫӢеӯҗпјҡint a = 100000, int b = -200000гҖӮ

longпјҡ

  • longж•°жҚ®зұ»еһӢжҳҜ64дҪҚгҖҒжңүз¬ҰеҸ·зҡ„д»ҘдәҢиҝӣеҲ¶иЎҘз ҒиЎЁзӨәзҡ„ж•ҙж•°пјӣ
  • жңҖе°ҸеҖјжҳҜ-9,223,372,036,854,775,808пјҲ-2^63пјүпјӣ
  • жңҖеӨ§еҖјжҳҜ9,223,372,036,854,775,807пјҲ2^63 -1пјүпјӣ
  • иҝҷз§Қзұ»еһӢдё»иҰҒдҪҝз”ЁеңЁйңҖиҰҒжҜ”иҫғеӨ§ж•ҙж•°зҡ„зі»з»ҹдёҠпјӣ
  • й»ҳи®ӨеҖјжҳҜ0Lпјӣ
  • дҫӢеӯҗпјҡ

floatпјҡ

  • floatж•°жҚ®зұ»еһӢжҳҜеҚ•зІҫеәҰгҖҒ32дҪҚгҖҒз¬ҰеҗҲIEEE 754ж ҮеҮҶзҡ„жө®зӮ№ж•°пјӣ
  • floatеңЁеӮЁеӯҳеӨ§еһӢжө®зӮ№ж•°з»„зҡ„ж—¶еҖҷеҸҜиҠӮзңҒеҶ…еӯҳз©әй—ҙпјӣ
  • й»ҳи®ӨеҖјжҳҜ0.0fпјӣ
  • жө®зӮ№ж•°дёҚиғҪз”ЁжқҘиЎЁзӨәзІҫзЎ®зҡ„еҖјпјҢеҰӮиҙ§еёҒпјӣ
  • дҫӢеӯҗпјҡ

doubleпјҡ

  • doubleж•°жҚ®зұ»еһӢжҳҜеҸҢзІҫеәҰгҖҒ64дҪҚгҖҒз¬ҰеҗҲIEEE 754ж ҮеҮҶзҡ„жө®зӮ№ж•°пјӣ
  • жө®зӮ№ж•°зҡ„й»ҳи®Өзұ»еһӢдёәdoubleзұ»еһӢпјӣ
  • doubleзұ»еһӢеҗҢж ·дёҚиғҪиЎЁзӨәзІҫзЎ®зҡ„еҖјпјҢеҰӮиҙ§еёҒпјӣ
  • й»ҳи®ӨеҖјжҳҜ0.0dпјӣ
  • дҫӢеӯҗпјҡdouble d1 = 123.4гҖӮ

booleanпјҡ

  • booleanж•°жҚ®зұ»еһӢиЎЁзӨәдёҖдҪҚзҡ„дҝЎжҒҜпјӣ
  • еҸӘжңүдёӨдёӘеҸ–еҖјпјҡtrueе’Ңfalseпјӣ
  • иҝҷз§Қзұ»еһӢеҸӘдҪңдёәдёҖз§Қж Үеҝ—жқҘи®°еҪ•true/falseжғ…еҶөпјӣ
  • й»ҳи®ӨеҖјжҳҜfalseпјӣ
  • дҫӢеӯҗпјҡboolean one = trueгҖӮ

charпјҡ

  • charзұ»еһӢжҳҜдёҖдёӘеҚ•дёҖзҡ„16дҪҚUnicodeеӯ—з¬Ұпјӣ
  • жңҖе°ҸеҖјжҳҜвҖҷ\u0000вҖҷпјҲеҚідёә0пјүпјӣ
  • жңҖеӨ§еҖјжҳҜвҖҷ\uffffвҖҷпјҲеҚідёә65,535пјүпјӣ
  • charж•°жҚ®зұ»еһӢеҸҜд»ҘеӮЁеӯҳд»»дҪ•еӯ—з¬Ұпјӣ
  • дҫӢеӯҗпјҡchar letter = вҖҳAвҖҷгҖӮ

Reference Types

Reference TypesжҳҜеӯҳеӮЁеӨҚжқӮзҡ„еҜ№иұЎзҡ„ ,жҜ”еҰӮиҜҙDateзұ» пјҢmail message зұ»

еңЁдҪҝз”ЁReference Types ж—¶пјҢжҲ‘д»¬еҝ…йЎ»жүӢеҠЁеҲҶй…ҚеҶ…еӯҳ

Primitive vs Reference Types

Primitive Types дёӯпјҢжҲ‘д»¬еЈ°жҳҺдёӨдёӘеҸҳйҮҸпјҢи®©д»–д»¬зӣёзӯүпјҢйӮЈд№ҲеҸӘжҳҜж•°еҖјдёҠзӣёзӯүпјҢе®ғд»¬зҡ„еӯҳеӮЁең°еқҖжҳҜдёҚдёҖж ·зҡ„

дҪҶжҳҜReference Types пјҢйҮҢйқўеӯҳеӮЁзҡ„ж—¶ең°еқҖпјҢжүҖд»Ҙи®© point2 = point1д№ӢеҗҺпјҢpoint1пјҢpoint2дёӯеӯҳеӮЁзҡ„жңүзӣёеҗҢзҡ„ең°еқҖгҖӮеҪ“еҜ№point1дёӯдҝ®ж”№ж—¶пјҢpoint2д№ҹдјҡзӣёеә”дҝ®ж”№гҖӮ
